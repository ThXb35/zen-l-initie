Notice d'utilisateur : 

	- Pour jouer en mode graphique uniquement, il suffit de double-cliquer sur le jar et de choisir la version graphique.

	- Pour jouer en mode console ou graphiquen il suffit de lancer le jar à l'aide de la commande "java -jar Herard_Zen.jar" et de lancer le mode graphique ou console.

La suite est très intuitive, il suffit d'entrer les chiffres corresponsant aux menus ou l'on souhaite aller. 

/!\ Attention à ne pas bouger le jar de son emplacement d'origine /!\
